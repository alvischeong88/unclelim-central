🔗 Connect to Google Sheets

This app syncs production & stock data directly into your Uncle Lim's Kitchen Report Google Sheet.

1Upload the Excel file to Google Sheets and copy the Sheet ID from the URL
2Go to Google Cloud Console → Enable Sheets API → Create Service Account → Download JSON key
3Share your Google Sheet with the client_email from the JSON (give Editor access)
4Paste the values below and tap Save

🔒 Your credentials are stored only in this device's local storage — never sent anywhere except directly to Google's API.
