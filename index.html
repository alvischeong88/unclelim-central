🔗 Connect to Google Sheets

Workers submit from this app → rows appear instantly in your Uncle Lim's Kitchen Report Google Sheet.

1Upload the Excel report to Google Sheets. Copy the Sheet ID from the URL between /d/ and /edit
2Go to console.cloud.google.com → Enable Google Sheets API → Credentials → Create Service Account → Keys → Add Key → JSON → Download
3Open the JSON file. Copy client_email and private_key values below
4In your Google Sheet → Share → paste the client_email → give Editor access
🔒 Credentials stored only on this device — sent directly to Google's API, never to any other server.
