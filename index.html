Connect to Google Sheets
1Upload Excel report to Google Sheets. Copy the Sheet ID from the URL (between /d/ and /edit)
2Go to console.cloud.google.com → Enable Sheets API → Credentials → Create Service Account → Keys → JSON → Download
3Share your Sheet with the client_email from the JSON → give Editor access
From URL: docs.google.com/spreadsheets/d/THIS_PART/edit
🔒 Credentials stored only on this device — never sent anywhere except Google's API directly.

💡 Tip: Open your JSON file in Notepad → find "private_key" → copy everything between the outer quotes (including -----BEGIN PRIVATE KEY----- and -----END PRIVATE KEY-----) → paste here.

Common issues:
❌ Wrong Sheet ID → paste full Google Sheet URL
❌ Sheet not shared → Share sheet with your service account email → give Editor access
❌ Key format wrong → open JSON in Notepad, copy private_key value including -----BEGIN...
❌ Wrong email → must end in .iam.gserviceaccount.com (not your Gmail)
